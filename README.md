# Methan0l

## Documentation
Available in the [Wiki Section](https://github.com/Hitonoriol/Methan0l/wiki)  

## How to use

### Build Methan0l

```
cd build && make
```  
(GCC 10+ is required)  

### Run Methan0l programs

**Execute** a program:  
```
./methan0l /path/to/src_file.mt0 [cmd_arg1 cmd_arg2 ...]
```  
\
**Interactive mode** (parse & execute expressions from stdin):  
```
./methan0l
```
